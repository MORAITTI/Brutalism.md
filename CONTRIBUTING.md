# Contributing to Brutalism.md

Thank you for your interest in contributing to this repository. This is a curated knowledge base about brutalism across architecture, design, music, fashion, and digital culture.

---

## What Can Be Contributed

- **New sections** — topics not yet covered (see ideas below).
- **Expanding existing sections** — adding buildings, designers, films, albums, books.
- **Translations** — all content exists in both English and Russian. Translations of new or existing content are welcome.
- **Images** — high-quality images embedded with `<img src="..."/>` tags.
- **Corrections** — factual errors, broken links, outdated information.
- **Sources** — adding citations to `sources.md` for claims made in existing content.

---

## File Structure

Each topic has its own folder. Inside each folder:

```
topic name/
├── topic name.md          ← English version
└── название темы.md       ← Russian version (Cyrillic)
```

Follow this convention for new content.

---

## Writing Style

- **Tone**: Encyclopedic, direct, and informative. Avoid promotional language.
- **Length**: Sections should be substantial — not single paragraphs. Use headers to organize.
- **Bilingual**: If you contribute in one language, note in your PR that a translation is needed.
- **Images**: Use `<img src="URL"/>` syntax. Prefer stable image URLs. Credit sources where possible.
- **Tables**: Use markdown tables for comparisons, timelines, and lists of works.
- **No fluff**: Brutalist writing for a brutalist topic — concise, honest, no padding.

---

## How to Contribute

1. **Fork** the repository.
2. **Create a branch** with a descriptive name: `add-brutalism-in-music` or `fix-timeline-dates`.
3. **Make your changes** following the file structure and style guide above.
4. **Update `sources.md`** if you are adding factual content that needs citation.
5. **Submit a Pull Request** with a clear description of what you added or changed.

---

## Ideas for New Content

If you are looking for inspiration, here are topics not yet fully covered:

- Brutalism in specific countries (Japan, Brazil, Yugoslavia, Poland)
- Brutalist photography as a genre
- Brutalist video games and game design
- Brutalist furniture and product design
- Brutalism in illustration
- Brutalist logos and brand identity
- Preservation case studies (specific saved or demolished buildings)
- Interviews or essays by brutalist architects

---

## Sources

When adding factual information, add your sources to `sources.md` in this format:

```
[Source Title](URL) — brief description of what it covers
```

---

## Questions

Open an Issue if you are unsure whether a contribution fits, or if you want to discuss a larger addition before investing the time.
